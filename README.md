## Project Purpose
CRUDZASO is not just a task manager; it is a robust **SPA (Single Page Application)** that implements a role-based access control (RBAC) system, ensuring that both administrators and end users operate in a secure, scalable, and visually stunning environment.

## Stack Technological
- **Core:** JavaScript Vanilla.
- **Tooling:** [Vite 5.0].
- **Backend:** [JSON Server].
- **Styles:** CSS3.
- **Router:** custom router.

## Elite Features

### Administration Module
- **Overview:** Dashboard with real-time analytics for the entire system.
- **Full Control:** Ability to monitor, edit, and delete tasks for any user.
- **Account Management:** Centralized administration of user profiles.
- **Metrics:** KPI visualization of overall project progress.

### User Module
- **Personal Dashboard:** Intuitive panel with personalized productivity metrics.
- **CRUD Management:** Full control over the lifecycle of your tasks.
- **Intelligent Prioritization:** Ranking by levels (Low, Medium, High).
- **Advanced Filters:** Segmentation by status (Pending, In Progress, Completed).
